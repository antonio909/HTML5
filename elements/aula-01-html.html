O elemento html O elemento html representa a raiz de um documento HTML. Os desenvolvedores são incentivados a especificar um atributo lang no elemento raiz html, indicando o idioma do documento. Isso auxilia as ferramentas de síntese de voz a determinar quais pronúncias utilizar, as ferramentas de tradução a definir quais regras aplicar, e assim por diante. Exemplo: O elemento html no exemplo a seguir declara que o idioma do documento é o inglês. Swapping Songs
